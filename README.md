# 🍽️ Canteen Management System

A simple web-based Canteen Management System built using **HTML, CSS, and JavaScript**.
Users can view food items and click to calculate the total bill dynamically.

---

## 🚀 Live Demo

👉 https://your-username.github.io/your-repo-name/

---

## 📌 Features

* 🥘 Displays canteen food items with images
* 💰 Shows price for each item
* 🖱️ Click items to add to total
* 🧮 Auto-calculates total bill
* 🎨 Simple and clean UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

To deploy:

1. Go to repository **Settings**
2. Open **Pages**
3. Select branch: `main`
4. Save and get your live link
